Préparation du deuxième *datasprint* pour le projet [ANR Portic](https://anr.portic.fr).

Plus d'informations dans les fichiers `/docs/**.md`.

# Installation

## Logiciels requis

- NodeJS
- Python (v3.10.1)

## Commandes

```
sh install.sh
```

# Démarrage

Construire et voir le site

```
npm run build
npm start
```

Ouvrir le notebook Jupyter

```
npm run notebook
```
